# ResearchRadar Slidev Deck

## Run

```bash
cd /home/dataset-local/ResearchRadar/presentations/researchradar-slidev
npm run dev
```

Open the URL printed by Slidev. The default is usually:

```text
http://127.0.0.1:3030
```

## Build

```bash
npm run build
```

## Export

Slidev export may download or require a browser runtime on first use.

```bash
npm run export:pdf
npm run export:pptx
```
